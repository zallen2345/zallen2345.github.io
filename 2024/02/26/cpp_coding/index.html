C++入门

在这一篇文章中,我会讲C++入门操作.
首先你必须有一个C++软件,我推荐Dev-c++但也可以用vscode（全英文）
vscode下载链接点这里.
如何安装就点这里.
接下来就是基础操作.
首先要认识一个东西叫做头文件,c++会拿一个头文件用于下面的代码.
样子是: #include<>,我们要用的是iostream,用它就可以用cin和cout,
分别是输入和输出. 下面是个例子:

它的输出是: hello,world! 注:要换行的话尾部加个endl或\n.
cout<<"hi!\n";或 cout<<"hi"<<endl;
下面是cin的例子:

#include<iostream>
using namespace std;
int main(){
   int n;
   cin>>n;
   cout<<n;
   return 0;
}
``` int n 表示定义一个int形式的n. cin>>n就是输入了,cout<<n就是输出.
这个就是一个输入什么数字,输出什么数字的程序.
